# My_Portfolio

Personal portfolio site for **Lester Leeroy Kandekande** — built with React, TypeScript, Vite, Bootstrap 5, and Framer Motion.

## Scripts

- `npm install` — install dependencies  
- `npm run dev` — local dev server  
- `npm run build` — production build to `dist/`  
- `npm run preview` — preview the production build  

## Deploy

For GitHub Pages, set `VITE_BASE=/My_Portfolio/` (or your repo name) when building, or use the workflow in `.github/workflows/deploy.yml`.
